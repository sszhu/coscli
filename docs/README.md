## Quick Start

1. **Install**: Run `../install.sh`
2. **Activate**: `source .venv/bin/activate`
3. **Configure**: `cos configure`
4. **Use**: `cos ls`

## Need Help?

- Run `cos --help` for command help
- Check [QUICK_REFERENCE.md](QUICK_REFERENCE.md) for examples
- Use `--debug` flag for troubleshooting
